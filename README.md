## Controls
- World Rotate - drag mouse
- World Zoom - rotate mouse wheel

## Actions
+ 🐔 Dwellers Move - drag Dweller with mouse
+ 🌿 Feed animals - drag Plant to Animal with mouse
+ 💰 Sell goods - click on the icons in the top menu
